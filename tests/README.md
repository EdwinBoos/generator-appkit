# Mocha - Tests


In order to run the tests you need to install mocha: <br>
``` npm install mocha -g ``` 

To run the app.test.js file you'll need to type: <br>
``` mocha app.test.js ```

To run the tests for a subgenerator
``` mocha controller.test.js ```
``` mocha fragment.test.js ```
``` mocha i18n.test.js ```
``` mocha view.test.js ```
